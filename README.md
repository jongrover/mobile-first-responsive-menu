# Mobile First Responsive Navigation (Unstyled)

A simple mobile first based responsive navigation that uses progressive enhancement so menu only appears if JavaScript is available, otherwise if no JavaScript is present users still have access to the essential navigation links. there is no additional style so you can add your own CSS.
